## Route 的功能

提供整个 umi 的路由处理，也是约定式路由的实现。

## 文件目录

|          文件名          |          作用          |           备注           |
| :----------------------: | :--------------------: | :----------------------: |
|         Route.ts         |      路由的核心类      |   封装了路由匹配等方法   |
|     routesToJSON.ts      | 路由转化为 json 的方法 |      用于前后端传递      |
| getConventionalRoutes.ts |      获取默认路由      | 前端开发时常用写的路由表 |
